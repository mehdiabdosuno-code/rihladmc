# S'TOURS DMC Morocco — Brand Guidelines

## Logo

The S'TOURS logo consists of:
- **Icon**: Red swirl triangle mark (`#A8371D`)
- **Wordmark**: "S'TOURS" in navy blue (`#1628A9`)
- **Tagline**: "DESTINATION MANAGEMENT COMPANY" in red (`#A8371D`)

### Logo files in this directory:
- `stours_dmc_logo.png` — Full logo (3508×2481px) for documents & presentations
- `stours_icon.png` — Icon only (400×464px) for favicons & compact use
- `stours_cover_template.png` — Official cover page template (774×1094px)
- `stours_footer.png` — Footer banner with 3 columns (1992×284px)
- `stours_bus.jpg` — Fleet photo for transportation sections

## Document Layout

### Cover Page
- Full-bleed image with S'TOURS logo in white
- Official cover template: `stours_cover_template.png`
- Travel designer adds editable second page with title, dates, route

### Header (every content page)
- Logo aligned right (150px wide)
- Red horizontal rule below

### Footer (every content page)
- Footer banner image: 3 columns (company info, Casablanca HQ, Marrakech office)
- Page numbers: "Reference — Page X / Y"

### Section Banners
- Full-width red (`#A8371D`) background
- White bold uppercase text
- Used for: General Information, Accommodation, Catering, Activities, etc.
